# Fuentes de datos

El archivo `data/conexiones.csv` contiene una columna llamada `fuente`.

En esta maqueta las distancias son referenciales. Para la entrega final se deben reemplazar por distancias verificadas y documentadas.

## Fuentes sugeridas para verificar distancias

- Google Maps
- Rome2Rio
- Distance.to
- ViaMichelin
- OpenStreetMap

## Formato recomendado

| Origen | Destino | Distancia km | Fuente |
|---|---|---:|---|
| Paris | Bruselas | 312 | URL o referencia consultada |
| Bruselas | Amsterdam | 209 | URL o referencia consultada |

## Recomendación

Mantener un criterio único: distancia terrestre aproximada en kilómetros. No mezclar kilómetros aéreos, tiempo de viaje, precio de pasajes y costo de combustible como pesos del grafo.
