# Pruebas de funcionamiento

Estas pruebas deben ejecutarse desde la interfaz gráfica.

## Prueba 1

- Origen: Paris
- Destino: Berlin
- Modo: Vehículo
- Perfil: Sedán
- Octanaje: 95

Resultado esperado:
- La aplicación debe mostrar una ruta óptima.
- Debe mostrar distancia total.
- Debe mostrar litros y costo estimado.
- Debe mostrar el grafo con la ruta resaltada.
